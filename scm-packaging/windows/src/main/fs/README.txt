SCM-Manager
===========

Requirements:

Be sure you have installed Java 8 or newer (11 is recommended) and the java.exe
is added to the PATH environment variable.

Installation:

Execute the following command to install the scm-server service.

scm-server install

If another user should be used for the service, the /p option can be used.

Start:

scm-server start

First Access
============

 URL:      http://localhost:8080
 Username: scmadmin
 Password: scmadmin
